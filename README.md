### Launch Codes
0. Clone repo

```
git clone git@github.com:Lauren-lee1/p01_spaghettiRat.git
```

1. In the repo, create a venv
```
python -m venv rat
source rat/bin/activate
```

2. Install packages
```
pip install -r requirements.txt
```

3. Start Flask server 
```
python app/__init__.py
```
